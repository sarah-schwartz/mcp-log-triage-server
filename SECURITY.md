# Security Policy

If you discover a security issue, please report it privately.

## Reporting

- If you have a private channel with the maintainers, use it.
- Otherwise, open a GitHub issue and request a private follow-up.

Please avoid sharing exploit details in public issues or pull requests.

## Scope

This repository is a local log triage server. Security concerns typically
relate to file access boundaries, logging of sensitive data, or dependency
vulnerabilities.
